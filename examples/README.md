# OpenDBS Examples

This directory contains examples of how to use OpenDBS.

## Basic Usage

See `basic-usage.js` for a simple example of creating a database, inserting documents, and querying them.

## Advanced Examples

- `fuzzy-search.js` - Fuzzy search example
- `bulk-insert.js` - Bulk insert operations
- `indexing.js` - Working with indexes
- `authentication.js` - User authentication and authorization
- `vector-search.js` - Vector similarity search

Run any example with:

```bash
node examples/<example-name>.js
```
